## 📌 About the Project

A collection of advanced performance analysis and telemetry tools for Rocket League. Each tool provides real-time tracking, visual overlays, and gameplay improvement features.

---

## 🔧 Available Tools

- **VISUALS.GG** — Clean and focused performance analyzer with detailed ESP for ball prediction, car positions, boost pads, and opponent mechanics.
- **PRISM** — Premium performance analyzer with intelligent AI-assisted mechanics, precise aerial and ground shot prediction, and smart boost management.
- **CROWN AI** — Premium AI-powered performance analyzer with advanced car control assistance, predictive aerials, smart boost management, and defensive positioning.
